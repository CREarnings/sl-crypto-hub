FIRST TIME SETUP
⚠️ One-time setup required
CR Earnings uses a GitHub Gist as a free database so all visitors see the same articles. This only needs to be done once.
How to set up:
1. Go to gist.github.com → New secret gist
2. Filename: cr-data.json · Content: {} → Click Create secret gist
3. Copy the Gist ID from the URL (long code after your username)
4. Go to GitHub → Settings → Developer settings → Personal access tokens (classic)
5. Generate token → tick gist and repo scopes → copy it
6. Enter your GitHub username and the repo name where this site is hosted
7. Fill in below 👇
